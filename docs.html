📡 统一网关
POST http://124.220.44.71/api/gateway

所有接口通过同一网关调用。请求通过 POST 表单或 JSON 提交，每次请求必须携带 token 和 api 两个公共参数。

📖 快速开始

1. 联系客服获取 Token
2. 选择 API 并确定业务参数
3. 调用网关: POST http://124.220.44.71/api/gateway
4. 解析返回 JSON

curl -X POST http://124.220.44.71/api/gateway -d "token=YOUR_TOKEN" -d "api=hs_list_main"
📦 通用响应格式
{
  "code": 20000,   // 20000=成功
  "msg": "success",
  "data": [...]    // 数据数组或对象
}
