列表与检索

证券代码表、新股日历、指数与多市场基础列表

stock_list

沪深京多市场股票合并列表。单条记录为一只证券的基础代码与名称，供后续查询作为入参。

UPDATE交易日盘后多时段刷新

沪深 A 股基础列表（不含北证合并）。代码、名称、交易所。

UPDATE交易日 ~16:20 后更新

新股申购与发行信息日历，按申购日期倒序。

UPDATE每日 ~17:00 后刷新
